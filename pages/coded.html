EleGaF (ELEgant GAme Framework) is a set of tools I am writing in Javascript to develop and share video game ideas quickly and easily.

KniDoRAn.

KniDoRAn (Knight Domination R Annealing) is a simple program that I wrote in R to estimate the minimal set of dominating knights on an NxN chess board. KniDoRAn uses a simulated annealing algorithm. I wrote KniDoRAn for an assignment in my Graph Theory course at St. Mary's College.
